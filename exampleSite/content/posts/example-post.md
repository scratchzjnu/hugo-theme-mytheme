## 介绍

欢迎使用My Theme！这是一个为Hugo静态网站生成器设计的现代、干净的主题。本主题具有响应式设计，支持深色模式，并包含了丰富的功能，适合个人博客、作品集和企业网站。

## 主题特性

- **响应式设计**：适配各种屏幕尺寸，从手机到桌面设备
- **深色模式**：自动检测系统偏好并支持手动切换
- **现代布局**：简洁、优雅的设计风格
- **丰富的组件**：包括导航栏、页脚、文章卡片等
- **良好的性能**：优化的CSS和JavaScript，快速加载
- **易于定制**：通过配置文件和部分模板进行定制

## 如何使用

1. **安装主题**：将主题文件夹复制到您的Hugo站点的`themes`目录中
2. **配置主题**：在`config.toml`或`hugo.toml`中设置`theme = "my-theme"`
3. **创建内容**：使用`hugo new content`命令创建新的内容文件
4. **启动服务器**：运行`hugo server -D`预览您的站点

## 自定义选项

您可以通过修改`config.toml`或`hugo.toml`文件来自定义主题的各种选项，包括：

- 网站标题和描述
- 菜单配置
- 社交媒体链接
- 首页CTA按钮
- 评论系统（Disqus）
- 谷歌分析
